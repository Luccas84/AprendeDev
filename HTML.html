Introdução ao HTML

HTML é a base de todos os sites que você visita na internet. Vamos aprender o essencial para começar a criar suas primeiras páginas web!

HTML em 10 segundos:

É a "estrutura" ou "esqueleto" de uma página web. Enquanto o CSS cuida da aparência e o JavaScript do comportamento, o HTML define o conteúdo e sua organização.

O que significa HTML?

HyperText Markup Language (Linguagem de Marcação de Hipertexto). Não é uma linguagem de programação, e sim uma linguagem de marcação que usa "tags" para estruturar conteúdo.

Estrutura Básica de um Documento HTML

Toda página HTML começa com esta estrutura essencial:

<!DOCTYPE html>
<html lang="pt-br">
<head>
  <meta charset="UTF-8">
  <title>Título da Página</title>
</head>
<body>
  <!-- Conteúdo visível da página -->
</body>
</html>

Explicando cada parte:

  • <!DOCTYPE html> - Define que é um documento HTML5
  • <html> - Elemento raiz que envolve todo o documento
  • <head> - Contém metadados e informações não visíveis
  • <body> - Contém todo o conteúdo visível da página

Curiosidade:

As tags normalmente vêm em pares (abertura e fechamento), como <html></html>. Algumas tags são auto-fechantes, como <img>.

Criando Conteúdo Básico

1. Títulos

HTML possui 6 níveis de títulos, do <h1> (mais importante) ao <h6> (menos importante).

<h1>Meu Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Seção Menor</h3>

2. Parágrafos

Use a tag <p> para criar parágrafos de texto.

<p>Este é um parágrafo de exemplo.</p>
<p>Outro parágrafo com mais texto.</p>

3. Links

Links são criados com a tag <a> (anchor) e o atributo href que define o destino.

<a href="https://www.exemplo.com">Visite nosso site</a>

Experimente!

Crie um arquivo com extensão .html (ex: meu-site.html), cole o código abaixo e abra no navegador:

<!DOCTYPE html>
<html>
<head>
  <title>Meu Primeiro Site</title>
</head>
<body>
  <h1>Olá, Mundo!</h1>
  <p>Este é meu primeiro parágrafo.</p>
  <a href="https://google.com">Buscar no Google</a>
</body>
</html>

Próximos Passos

Agora que você já conhece o básico do HTML, pode:

  1. Praticar criando pequenas páginas
  2. Aprender mais tags HTML (listas, imagens, tabelas)
  3. Começar a estudar CSS para estilizar suas páginas

Dica Importante:

Sempre verifique se suas tags estão corretamente abertas e fechadas. Erros comuns incluem esquecer a barra em tags de fechamento ou usar maiúsculas em tags (HTML é case-sensitive para nomes de tags).
