Introdução ao JavaScript

JavaScript é a linguagem que dá vida às páginas web! Enquanto HTML estrutura e CSS estiliza, o JavaScript adiciona interatividade e comportamento dinâmico.

JavaScript em 10 segundos:

É a linguagem de programação que roda no navegador e permite criar páginas web interativas, desde simples animações até complexas aplicações web.

O que JavaScript pode fazer?

  • Adicionar interatividade a páginas web
  • Manipular HTML e CSS dinamicamente
  • Validar formulários antes do envio
  • Criar animações e efeitos visuais
  • Carregar dados sem recarregar a página (AJAX)
  • Desenvolver aplicações web complexas

Como adicionar JavaScript a uma página

1. JavaScript interno (na página HTML)

<script>
  // Seu código JavaScript aqui
</script>

2. JavaScript externo (arquivo .js separado)

<script src="scripts.js"></script>

Melhor prática:

Para projetos reais, use sempre arquivos externos. Mantém seu código organizado e permite reutilização.

Conceitos Básicos

1. Variáveis

Armazenam dados que podem ser usados e modificados durante a execução do programa.

// Declarando variáveis
let mensagem = "Olá, mundo!";
const PI = 3.14;
var idade = 25;

// Usando variáveis
console.log(mensagem); // Mostra "Olá, mundo!"

2. Funções

Blocos de código que realizam tarefas específicas e podem ser reutilizados.

// Definindo uma função
function cumprimentar(nome) {
  return "Olá, " + nome + "!";
}

// Chamando a função
let saudacao = cumprimentar("João");
console.log(saudacao); // "Olá, João!"

3. Eventos

Ações que ocorrem na página (cliques, teclas pressionadas, etc.) e podem ser capturados para executar código.

Demonstração Interativa

Você clicou no botão! Isso é JavaScript em ação.
// Código por trás do botão acima
function mostrarMensagem() {
  let mensagem = document.getElementById("demoMessage");
  mensagem.style.display = "block";
}

Exemplo Completo

Vamos criar um contador simples:

Contador: 0

// JavaScript
let contador = 0;

function incrementar() {
  contador++;
  document.getElementById("contador").textContent = contador;
}

function zerar() {
  contador = 0;
  document.getElementById("contador").textContent = contador;
}

Experimente!

Crie um arquivo HTML com este código para ver o JavaScript em ação:

<!DOCTYPE html>
<html>
<head>
  <title>Meu Primeiro JavaScript</title>
</head>
<body>
  <button onclick="alert('Olá, mundo!')">Clique Aqui</button>
  <script>
    function mudarCor() {
      document.body.style.backgroundColor = "#f0f0f0";
    }
  </script>
  <button onclick="mudarCor()">Mudar Cor de Fundo</button>
</body>
</html>
