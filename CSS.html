Introdução ao CSS

Se HTML é o esqueleto de uma página web, o CSS é a roupa que deixa essa página bonita e estilosa! Vamos aprender como transformar páginas simples em trabalhos visuais atraentes.

CSS em 10 segundos:

CSS (Cascading Style Sheets) é uma linguagem que controla a aparência do conteúdo HTML. Enquanto o HTML estrutura, o CSS estiliza.

O que significa CSS?

Cascading Style Sheets (Folhas de Estilo em Cascata). O "cascata" refere-se à forma como os estilos são aplicados - como uma cascata de regras que fluem de cima para baixo.

Como o CSS funciona?

O CSS usa um sistema de regras para aplicar estilos aos elementos HTML. Cada regra consiste em:

seletor { propriedade: valor; }

/* Exemplo: */
h1 {
  color: blue;
  font-size: 24px;
}

Explicando:

  • Seletor: Indica qual elemento HTML será estilizado (h1, p, .classe, #id)
  • Propriedade: O que você quer mudar (cor, tamanho, margem, etc.)
  • Valor: Como você quer que fique (azul, 20px, etc.)

3 Formas de Adicionar CSS

1. CSS Inline (no próprio elemento)

<h1 style="color: red; font-size: 30px;">Título</h1>

2. CSS Interno (na seção <head>)

<style>
  p {
    color: green;
  }
</style>

3. CSS Externo (arquivo .css separado)

<link rel="stylesheet" href="estilos.css">

Melhor prática:

Use CSS externo sempre que possível. Mantém seu código organizado e permite reutilizar estilos em várias páginas.

Exemplos Práticos

1. Mudando cor de fundo

body {
  background-color: #f0f0f0;
}

Esta caixa tem cor de fundo #f0f0f0

2. Alterando fontes

p {
  font-family: Arial, sans-serif;
  font-size: 16px;
  color: #333;
}

Este texto está com fonte Arial, tamanho 16px e cor #333

3. Centralizando elementos

.centro {
  text-align: center;
  margin: 0 auto;
  width: 50%;
}
Este elemento está centralizado

Experimente!

Crie um arquivo CSS (ex: estilos.css) e vincule a uma página HTML:

/* estilos.css */
body {
  font-family: Arial;
  background-color: #f9f9f9;
}

h1 {
  color: #4a6fa5;
  text-align: center;
}

.destaque {
  background-color: yellow;
  padding: 10px;
}

Próximos Passos

Agora que você já conhece o básico do CSS, pode:

  1. Praticar modificando páginas existentes
  2. Aprender sobre seletores mais avançados
  3. Explorar propriedades como margin, padding e display
  4. Começar a estudar layouts com Flexbox e Grid

Dica Importante:

Use ferramentas como o Inspetor do navegador (F12) para experimentar com CSS em tempo real em qualquer site!
